# Wildfire_Segment_from_Remote_Sensing_Data
This repo provides the DL model checkpoints for wildfire segmentation from Sentinel-2 data. The input images should be in RGB composition with the bands of Green, NIR, and SWIR.
The model checkpoint is available at: https://drive.google.com/file/d/1ykFf67hrOigyWrZFTDkFf5LmFh-384Lo/view?usp=sharing
